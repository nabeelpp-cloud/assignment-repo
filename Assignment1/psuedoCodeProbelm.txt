7 - Find the largest and smallest element in an array

Start
Input array
largest = array[0]
smallest = array [0]
For i from 1 to End off array do
	if array[i] > largest then
		largest = array[i]
	else if array[i] < smallest
		smallest = array[i]
print largest , smallest
End

10 - Find the second largest number in a list

Start
Input list
Largest = list [0]
SecondLargest = list [0]
For i from 1 to End of list do
	if list[i] > Largest
		SecondLargest = Largest
		Largest = list[i]
print SecondLargest
Stop
